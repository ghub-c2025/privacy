Privacy Policy

Last Updated: June 10, 2024

Information We Do Not Collect

The GitHub Repo Pusher App does not collect, store, or transmit:

  • Your GitHub username or password
  • Your personal files or repository data
  • Any tracking, analytics, or personal data

How the App Works

  • The App requires a GitHub Personal Access Token (PAT) to push files to your repositories
  • Your token is stored locally on your device and is never transmitted to external servers
  • The App only interacts with GitHub's API for actions you explicitly request
  • All operations occur directly between your device and GitHub's servers

No Ads or Tracking

  • The App contains no advertisements of any kind
  • No third-party trackers or analytics services are used
  • We do not monetize your data in any way

Security

  • Your GitHub token is stored securely using Android's encrypted storage mechanisms
  • The App cannot access your GitHub account beyond the specific permissions granted by your token
  • You can revoke the token anytime through your GitHub account settings
  • We recommend using tokens with the minimum required permissions

Changes to This Policy

We may update this Privacy Policy occasionally to reflect changes in our practices. When we do, we'll revise the "Last Updated" date at the top of this page. We encourage you to review this policy periodically.
